## SOFTWARE DEVELOPMENT PORTFOLIO

This repository contains foundational software development projects built with Python, Tkinter, and C++. The projects demonstrate GUI design, event-driven programming, console-based application logic, file handling, input validation, modular code structure, and version control with Git and GitHub. 

## 📌 Project Highlights

| Project | Technology | Description |
|---|---|---|
| [Digital Calculator](python/digital-calculator) | Python, Tkinter | Desktop calculator application with arithmetic operations, keyboard input, safe expression evaluation, and error handling |
| [Digital Clock](python/digital-clock) | Python, Tkinter | Desktop clock application that displays the current time, weekday, and full date through an automatically updating GUI |
| [ATM Management System](cpp/atm-management-system) | C++ | Console-based ATM application with user authentication, balance management, deposits, withdrawals, transaction history, and file handling |
